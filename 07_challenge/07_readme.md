Описание ошибки:
Функция была вызвана с fizzbuzz(99), из-за чего цикл range(1, max_num) не доходил до числа 99 включительно.

Как исправил:
Заменил аргумент на 100, чтобы цикл обрабатывал числа от 1 до 99.

Что понял:
У range() верхняя граница не включается, поэтому для диапазона 1–99 нужно передавать 100.
