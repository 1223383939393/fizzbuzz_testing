Описание ошибки:
Метод fizzbuzz был объявлен без параметра self, хотя вызывался через объект класса. Из-за этого возникала ошибка при передаче аргумента.

Как исправил:
Добавил self в определение метода fizzbuzz(self, max_num).

Что понял:
Методы класса, вызываемые через объект, должны принимать self первым параметром. Это нужно, чтобы метод мог работать с экземпляром класса.

Как работает класс и метод:
Класс Fizz_Buzz задаёт параметры для задачи FizzBuzz — делители 3 и 5 и строки fizz и buzz. После создания объекта метод fizzbuzz(self, max_num) перебирает числа от 1 до max_num - 1 и выводит fizzbuzz, если число делится на оба делителя, fizz, если делится только на 3, и buzz, если делится только на 5. Объект создаётся в блоке if __name__ == '__main__':, и метод вызывается с аргументом 100, чтобы обработать числа от 1 до 99.
