Описание ошибки:
В коде была использована неверная переменная из модуля conf: conf.num вместо conf.num2. Из‑за этого при обращении к конфигу возникала ошибка атрибута.

Как исправил:
Заменил conf.num на conf.num2, чтобы использовать правильное значение второго делителя.

Что понял:
Важно аккуратно работать с внешними модулями и конфигами: имя переменной должно совпадать с тем, что объявлено в conf.py. Любая опечатка приводит к ошибке при выполнении программы.
