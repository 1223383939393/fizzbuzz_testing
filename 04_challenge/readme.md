Описание ошибки:
Внутри блока if была нарушена структура отступов: команда print() оказалась вне тела условия, из-за чего возникала ошибка.

Как исправил:
Исправил отступы и разместил print() внутри блока if.

Что понял:
В Python отступы очень важны, потому что они задают структуру кода. Если отступ сломан, программа не запустится.

Как работает функция:
Функция fizzbuzz(max_num) перебирает числа от 1 до max_num - 1 и для каждого числа проверяет делимость на 3 и 5. В случае делимости на оба делителя она выводит fizzbuzz, при делимости только на 3 выводит fizz, при делимости только на 5 выводит buzz.
