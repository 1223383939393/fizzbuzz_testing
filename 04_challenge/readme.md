Описание ошибки:
Внутри блока if была нарушена структура отступов: команда print() оказалась вне тела условия, из-за чего возникала ошибка.

Как исправил:
Исправил отступы и разместил print() внутри блока if.

Что понял:
В Python отступы очень важны, потому что они задают структуру кода. Если отступ сломан, программа не запустится.
